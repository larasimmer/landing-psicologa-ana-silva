# Landing Page - Psicóloga Ana Silva

Este projeto foi desenvolvido com foco em demonstrar habilidades de criação de landing pages institucionais voltadas à conversão, aplicando boas práticas de UX, acessibilidade e SEO, comuns em projetos reais para profissionais liberais e pequenos negócios.


## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3**: Estilização moderna, layouts flexíveis (Flexbox) e animações suaves.
- **JavaScript (Vanilla)**: Lógica para o menu hamburger e sincronização do slider de depoimentos.
- **Font Awesome**: Ícones vetoriais para melhor acessibilidade e design.
- **Google Fonts**: Tipografia otimizada (Poppins e Instrument Sans).

## Principais Funcionalidades

- **Mobile-first & Design Responsivo**: Layout desenvolvido priorizando dispositivos móveis, com adaptação fluida para tablets e desktop.
- **Menu Hamburger**: Navegação otimizada para mobile com menu lateral intuitivo.
- **Slider de Depoimentos**: Implementado com **CSS Scroll Snap** para uma experiência de deslize nativa e fluida em dispositivos touch.
- **Acessibilidade (A11y)**: Uso de atributos ARIA, labels em formulários e ícones decorativos ocultos para leitores de tela.
- **Formulário de Contato**: Estruturado com validação básica e focado em conversão.
- **Botão Flutuante de WhatsApp**: Acesso rápido para contato direto.

## Estrutura do Projeto

- `index.html`: Estrutura semântica com headings hierárquicos, uso correto de section, nav, main, além de meta tags e foco em performance e legibilidade para mecanismos de busca.
- `style.css`: Estilos globais, variáveis e consultas de mídia (media queries).
- `script.js`: Lógica interativa do front-end.
- `images/`: Diretório contendo logotipos, fotos e favicon.

## Como Executar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para uma experiência de desenvolvimento melhor, recomenda-se o uso de uma extensão como "Live Server" no VS Code.
